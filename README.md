# Frank's Outfit Game

A web game where Frank the tailor tries to guess the outfit you have in mind. Think of an outfit, and Frank will use his knowledge of fashion and logic to figure out what you're wearing.

## How to Play

1. Think of an outfit in your mind (e.g., "red dress", "blue jeans and t-shirt")
2. Frank will ask you yes/no questions about your outfit
3. Answer honestly to help Frank narrow down the possibilities
4. Frank will make a guess when he's confident enough
5. See how few questions it takes Frank to guess your outfit!

## Tech Stack

- **Phaser 3** - Game engine for rendering and game logic
- **Vite** - Build tool and development server
- **Howler.js** - Audio library for sound effects and music
- **localStorage** - Save game progress and high scores

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
